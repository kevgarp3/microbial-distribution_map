# README: Metadata

all of the files specified below. However, these files are not being publicly shared as they directly derive from the list containing 93,690 different identifiers of skin microbiome-related sequencing Bio-Projects from ENA which is currently (March 2026) under active research (please refer to the README under the data/ directory).
