# README: Raw Data

As the actual list containing 93,690 different identifiers of skin microbiome-related sequencing Bio-Projects from ENA is currently (March 2026) under active research, it cannot be publicly disclosed. However a mock file containing only the necessary identifiers for any user to reproduce the available Krona plots in this repository by cloning it and executing the Snakemake workflow.

To inquire about the usage of the complete list of 93,690 sequencing project identifiers, please reach out to me so that I can assess sharing the contact of the researcher responsible of authorizing its release. Thank you!
